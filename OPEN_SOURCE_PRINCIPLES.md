## üìú Core Principles

### 1. **Open Source License**
- **MIT License** (maximum freedom, no restrictions)
- Allows commercial use, modification, distribution
- No copyleft restrictions
- Maximum adoption and community growth

### 2. **Open Standards Compliance**
- **W3C Standards:** RDF, OWL, SPARQL, JSON-LD
- **Interoperability:** Export/import to all major formats
- **Portability:** No vendor lock-in
- **Standards-First:** Follow semantic web best practices

### 3. **Community Ownership**
- **Community-Driven Development:** Open to contributions
- **Transparent Roadmap:** Public development plans
- **Open Governance:** Community input on direction
- **No Single Control:** Distributed decision-making

### 4. **No Vendor Lock-In**
- **Export to Any Format:** RDF, OWL, JSON-LD, Neo4j, etc.
- **Self-Hosted Option:** Can run entirely independently
- **No Proprietary Features:** All features open-source
- **Portable Data:** Knowledge graphs portable across systems

### 5. **Transparency**
- **Public Development:** All code public, all issues public
- **Open Roadmap:** Public feature planning
- **Transparent Funding:** All funding sources public
- **Open Communication:** Public discussions and decisions

## üö´ What We Will NOT Do

### 1. **No Proprietary Features**
- **Never:** Create closed-source "enterprise" features
- **Never:** Require proprietary licenses for core features
- **Never:** Lock users into proprietary formats
- **Commitment:** All features remain open-source

### 2. **No Vendor Lock-In**
- **Never:** Require proprietary services
- **Never:** Lock data into proprietary formats
- **Never:** Force users to use specific vendors
- **Commitment:** Always provide open alternatives

### 3. **No Corporate Control**
- **Never:** Sell to a single corporation that would close-source it
- **Never:** Accept funding with proprietary strings attached
- **Never:** Allow single entity to control direction
- **Commitment:** Community ownership and control

### 4. **No Hidden Costs**
- **Never:** Charge for core features
- **Never:** Require paid licenses for essential functionality
- **Never:** Hide costs or restrictions
- **Commitment:** Transparent, open, free
