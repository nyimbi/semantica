4. GraphRAG Architecture

Hybrid Retrieval Process

  1. Vector Search: Find top-K similar chunks using embeddings
  2. Entity Extraction: Extract entities from retrieved chunks
  3. Graph Expansion: Traverse knowledge graph from seed entities (2-3 hops)
  4. Hybrid Fusion: Combine vector and graph results using reciprocal rank fusion
  5. Reranking: Score and rank final results
  6. Context Building: Construct structured context with citations
  7. Answer Generation: Generate answer using LLM with graph-enhanced context

Result: Significant accuracy improvement over vector-only RAG
